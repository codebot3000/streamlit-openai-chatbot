# Introduction 
Streamlit test application for testing OpenAI LLM calls using chat completions or LangChain and providing a system prompt. 

# Getting Started
Clone code locally.
```powershell
git clone 
```

# Build and Test
Run application locally using:
```powershell
# PowerShell
streamlit run chatplus.py
```
